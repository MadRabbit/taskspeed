You can exclude any library or browsers results by passing an ?exclude= parameter to the url. The value is a "+" joined list of the id's as they happen to be. eg: to disable IE 8.0 and Firefox 2 add ?exclude=ie80+firefox20 ... The id's are generated based on a regexp and how php get_browser describes a particular useragent, or the library name stripped of all complex characters. eg: Dojo 1.3.0 -> dojo130
